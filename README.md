# Transport-app
Full-stack passenger transport booking app built with React, Node.js, Express, MongoDB Atlas, and JWT authentication.

A full-stack Transport Booking Web Application where users can browse routes, 
book seats, and track their bookings. Admins can manage vehicles, routes, and 
all bookings through a dedicated dashboard.

Built with React + Tailwind CSS (frontend), Node.js + Express (backend), 
MongoDB Atlas (database), and JWT-based role authentication (user/admin).
